# DNA Sequence Alignment

## Description
Implementation of sequence alignment algorithms to compare DNA sequences.

## Algorithms Used
- Needleman–Wunsch (Global Alignment)
- Smith–Waterman (Local Alignment)

## Features
- Sequence comparison
- Performance analysis
- Alignment scoring

## Technologies
- Python
- Jupyter Notebook

## Insights
Smith-Waterman is faster for local alignment, while Needleman-Wunsch aligns full sequences.
